引言： 异常也是一种类，对异常的处理就是对异常类的对象进行处理

## 异常
是一种类  **家族图：**  
【Throwable】异常类的顶级父类  
-子类 | 【Error】-程序员解决不了 | 【Exception】-程序中的异常
【Exception】
-子类 | 【RuntimeException】 运行时异常
* 【Error】、【RuntimeException】类及其子类，异常不做处理系统不会提示  
    【其他异常类】当可能出现时，系统会爆红，出于善意强制你做处理，  
     如果你很确定自己不会出问题，可以一路throw给jvm

```java
    > 异常为什么处理？
     -当程序运行时，会出现很多“动态数据”造成的jvm虚拟机强制停止程序
      我们要避免这种情况的出现

      *异常是可能出现，对异常的处理是对异常的预判
------------------------------------------------------------------
    > 异常的处理
     # 直接解决，很多需要立即解决的小问题 | 也是汇聚的异常最终的处理方式
        `try-catch-finally` 结构 -> 捕获处理，快捷是`选中目标+ctrl+alt+t`

        try{
            // 可能会出现异常的代码
            语句1   |按顺序执行，直到某个语句出现异常，则下面的语句不再执行
            语句2       直接跳转到对应的【catch】语句,然后跳出
                        `try-catch-finally` 结构,向下执行
                    |如果没有异常出现，直接跳过`try-catch-finally` 结构，
                        向下执行
        }catch (异常对象(*一般命名对象名为 e*)){
            // 我们一般会在这里把异常写入日志
            e.printStackTrace(); -> 打印异常具体信息
            e.getMessage().sout -> 打印异常的类型
        }catch(异常类 e){
            // 精确的捕获对应的类型
        }catch(Exception e){
            // 用父类Exception进行兜底，保证出现任意类型都能被捕获
        }finally{
            // 在【try-catch】之后执行，无论【try-catch】结构是否有触法
            // 且即使有触法抛出异常，他也会执行
            //我们一般用于释放在【try-catch】结构中被占用的资源
            //只有当程序先他一步终止，才能阻止他执行
        }

     # 暂缓处理，异常统一往上抛 | 一般是大问题往上抛
        `throw 【异常对象】` -> 该语句执行时，目前的方法不会再往下执行，
        同时方法声明可以加入`throws 【异常类型】` -> 告诉调用处这个方法可能存在的异常
        [方法声明][throws 类型1,类型2][方法体] 

        * 此时执行逻辑是：
        异常链：throw1，从方法抛出 -> 调用处接住，捕获处理/throw抛出
        -> 如果抛出，继续向上 -> 汇聚到main方法调用处，捕获处理/throw抛出->
        此时，main方法如果抛出，则是抛给虚拟机，导致程序强制停止
            这条处理异常的线只有1【在调用的顶部被捕获】/2【jvm强制终止程序】的结果
            （*我们要避免情况2的出现*）   

     * 无论方法有无异常，只要用捕获/抛出 处理了当前的方法，
          此时，在其他方法调用这个了做处理的方法，那这个方法也要进行捕获/抛出 的处理
          形成*异常链
--------------------------------------------------------------------------           > 自定义异常
    # 构建
        就是构建一个【类】继承【Exception】
        构造器可以使用`public 【类】(String Message)`-> 传入异常信息

    # 触发
        必须配合【分支结构】【throw】实现    
        if(条件){
            throw new 【类】();
        }
```
